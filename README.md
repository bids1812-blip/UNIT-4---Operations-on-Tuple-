Theory: -
A tuple in Python is a built-in data structure used to store multiple elements in a single variable. Tuples are ordered and immutable, meaning their elements cannot be changed once created. 

Operations on Tuple: -
1.	Tuples are created using round brackets (). 
2.	Tuples cannot be added, removed, or modified directly. 
3.	Two or more tuples can be combined using the + operator. 
4.	The number of values in a tuple can be found out using len() function. 
Tuples are useful when data should remain constant throughout the program execution.
